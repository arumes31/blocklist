Fail2ban

Configure Fail2ban to push bans to this app. Add an action file /etc/fail2ban/action.d/blocklist.conf:

[Definition]
actionban = curl -X POST -H "Content-Type: application/json" -d '{"ip":"<ip>","act":"ban","reason":"fail2ban-<name>","username":"{{ .admin_user }}","password":"YOUR_PASSWORD"}' {{ .base_url }}/api/v1/webhook
actionunban = curl -X POST -H "Content-Type: application/json" -d '{"ip":"<ip>","act":"unban","username":"{{ .admin_user }}","password":"YOUR_PASSWORD"}' {{ .base_url }}/api/v1/webhook

Examples

Curl Examples

List Blocked IPs (JSON)

curl -H "Authorization: Bearer YOUR_TOKEN" \
     {{ .base_url }}/api/v1/ips_list

Block an IP (Webhook style)

curl -X POST {{ .base_url }}/api/v1/webhook \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"ip": "1.2.3.4", "act": "ban", "reason": "manual"}'

Unblock an IP (Webhook style)

curl -X POST {{ .base_url }}/api/v1/webhook \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"ip": "1.2.3.4", "act": "unban"}'

Add IP to Whitelist (Webhook style)

curl -X POST {{ .base_url }}/api/v1/webhook \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"ip": "1.2.3.4", "act": "whitelist", "reason": "trusted service"}'

Wget Examples

Download Blocklist (TXT)

wget --header="Authorization: Bearer YOUR_TOKEN" \
     {{ .base_url }}/api/v1/ips/export?format=txt \
     -O blocklist.txt

Block an IP (POST)

wget --header="Authorization: Bearer YOUR_TOKEN" \
     --header="Content-Type: application/json" \
     --post-data='{"ip":"1.2.3.4","act":"ban"}' \
     {{ .base_url }}/api/v1/webhook -O -

Unblock an IP (POST)

wget --header="Authorization: Bearer YOUR_TOKEN" \
     --header="Content-Type: application/json" \
     --post-data='{"ip":"1.2.3.4","act":"unban"}' \
     {{ .base_url }}/api/v1/webhook -O -

Add IP to Whitelist (POST)

wget --header="Authorization: Bearer YOUR_TOKEN" \
     --header="Content-Type: application/json" \
     --post-data='{"ip":"1.2.3.4","act":"whitelist","reason":"trusted"}' \
     {{ .base_url }}/api/v1/webhook -O -
