Fail2ban

Configure Fail2ban to push bans to this app. Add an action file /etc/fail2ban/action.d/blocklist.conf:

[Definition]
actionban = curl -X POST -H "Content-Type: application/json" -d '{"ip":"<ip>","act":"ban","reason":"fail2ban-<name>","username":"{{ .admin_user }}","password":"YOUR_PASSWORD"}' {{ .base_url }}/webhook
actionunban = curl -X POST -H "Content-Type: application/json" -d '{"ip":"<ip>","act":"unban","username":"{{ .admin_user }}","password":"YOUR_PASSWORD"}' {{ .base_url }}/webhook
